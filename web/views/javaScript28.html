window 객체

window객체는 자바스크립트 최상위 객체이며 BOM과 DOM으로 나뉜다.

BOM(Browser Object Model) : location객체, navigation객체, history객체, screen객체

DOM(Document Object Model) : document객체 (태그라고 생각..)

screen 객체

웹 브라우저 화면이 아닌 운영체제 화면의 속성을 가지는 객체이다.

location객체

브라우저의 주소 표시줄과 관련된 객체이다.



navigator객체
